## 디자인 원칙
- 각 섹션은 독립적인 div로 구성 (스크롤 시 섹션별로 보임)
- 다크/라이트 모드 모두 지원 (기존 CSS 변수 사용)
- 섹션 간 구분: 배경색 교차 또는 여백으로 처리
- 기존 카드 3개 UI 완전히 제거
- 미니멀하고 프로페셔널한 느낌
- 태그 (NEWS / AI / ADS): accent 색상 작은 배지
- 텍스트 위계: 태그 → 제목(크게) → 설명(보통) → 리스트(작게)
- 섹션 3,4,5 좌우 이미지 자리에는
  심플한 SVG 아이콘 또는 숫자/텍스트 강조 요소로 대체
  (실제 이미지 없이도 시각적 균형 유지)
